Il codice spiegato

void setup() {
  pinMode(13, OUTPUT); // Pin LED come uscita
}

void loop() {
  digitalWrite(13, HIGH); // LED acceso
  delay(1000);            // Attendi 1 sec
  digitalWrite(13, LOW);  // LED spento
  delay(1000);            // Attendi 1 sec
}

Questo codice accende e spegne un LED ogni secondo usando Arduino UNO.
